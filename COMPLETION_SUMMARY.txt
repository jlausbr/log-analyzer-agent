                              RESUMO EXECUTIVO
================================================================================

Este projeto implementa um agente inteligente de análise de logs usando LangGraph
e OpenAI GPT-4. O agente automatiza a análise de arquivos de log, identifica
padrões de erro e gera relatórios técnicos estruturados em Markdown.

STACK TECNOLÓGICO:
✅ Python 3.10+
✅ LangGraph (orquestração de agentes)
✅ LangChain + OpenAI GPT-4 (análise inteligente)
✅ Pydantic (validação de dados)
✅ pytest (testes unitários)

AGENTE LANGGRAPH (5 NÓS):

  4. analyze_with_llm()  - Análise com GPT-4
  5. generate_report()   - Gera relatório Markdown

FERRAMENTAS CUSTOMIZADAS:

     - Categoriza por nível (ERROR, WARNING, INFO, OTHER)
